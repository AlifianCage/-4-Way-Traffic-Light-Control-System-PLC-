# -4-Way-Traffic-Light-Control-System-PLC-
⚙️ System Description

The system controls traffic lights for a four-way intersection, ensuring that only one direction has a green signal at a time while others remain red.

Each direction follows a sequence:

Green light (vehicles allowed to pass)
Yellow light (warning phase)
Red light (stop)

The system operates in a continuous loop, switching between directions using timers to maintain smooth and safe traffic flow.

🔌 Inputs & Outputs

Inputs:
- System Start Button
- System Stop Button (Hold 3 Second)
- Emergency Stop 

Outputs:
- North Traffic Light (Red, Yellow, Green)
- South Traffic Light (Red, Yellow, Green)
- East Traffic Light (Red, Yellow, Green)
- West Traffic Light (Red, Yellow, Green)

🖥️ Tools & Technologies
PLC Programming (Ladder Diagram)
Omron CX Programmer V9.7 / CX Designer
Timer-Based Control Logic
